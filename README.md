# CityLife
Metropolis-2D : Interactive 2D Cityscape in C++/OpenGL (GLUT). Features a dynamic day/night cycle, animated Metro rail, and urban traffic. Real-time speed controls and environment toggles via keyboard/mouse. Showcases 2D transformation matrices and smooth timer-based animation. Built for  CSE Computer Graphics.
Key Features
Day/Night Transition: Real-time environment toggle. The cityscape shifts from vibrant blue skies to deep navy, with building windows lighting up and vehicle headlights activating automatically at night.

Animated Traffic System: Features four distinct moving entities—a Metro train on a flyover, a bus, a cargo truck, and a car.

Physics-Based Rotation: Vehicle wheels utilize trigonometric functions to simulate realistic rotation relative to their movement speed.

Procedural Architecture: Includes 9 unique building designs, a flagpole with the national flag, and a garden landscape rendered using primitive shapes and polygons.

Interactive Controls: Users can manipulate the simulation speed and pause/resume individual animations using keyboard and mouse inputs.
Input,Action
D / N,Toggle Day / Night Mode
UP / DOWN,Adjust Metro Train Speed
LEFT / RIGHT,Adjust Sun or Moon Speed
SPACE,Master Pause/Resume All Animations
Mouse Click,Pause/Start specific vehicles or celestial bodies
